    • How to display "Hello World" in Python 3.8.5

    -----------------------------------

    This will teach you how to display "Hello World" in the programming language called Python.

    ---------------------------------------

    ---------------------------------------

    First things first: download Python

    Go to python.org > Download and follow the steps to set up Python on your computer

    -----------------------------------------

    ONCE YOU'VE INSTALLED PYTHON:

    We are going to learn the simplist function in Python: print().

    New Function:

    print(string, integer, variable...)

    String (str) example: 'Hello world'

    Integer (int) example: 6

    ---------------------------------------

    This code will display "Hello, World!":

    print("Hello, World!")

    ---------------------------------------

    But what if I want to print a text with a variable...

    REALLY EASY!!!(at least the printing function is)

    i = "hello world"

    print(i)

    Output: hello world

    n = "goodbye world"

    i = "goodbye world"

    Output:

    hello world

    print(n)

    goodbye world
