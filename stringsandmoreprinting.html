Strings and More Printing

Remember how we learned what strings are and the print() function? In this lesson, we are going to display more text.

--------------------------------------------------

Let's start with print('Asheng'). The output will be Asheng If you got an error, check if your code looks like this: print(Asheng). It will say that Asheng is not defined. Why? A string needs to have quotation marks around it. Without quotation marks, it's a variable. Unless if you do this:

Asheng = 'Asheng'

print(Asheng)
