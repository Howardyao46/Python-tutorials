Your First Big Python Project!

We are going to display some text to "remake" this website.

Before we get started, you can't do this in the Python shell. You must use an IDE for this project. We are going to use repl.it. Repl.it is free, and you can also do "multiplayer" with other people. What does that mean? You can invite other people into a project and you can see them edit and where their cursors are. So now, go to repl.it and sign up. Click on "new repl," select "Python," and then begin coding!

Begin typing:

print('How to display "Hello World" in Python 3')

If you press run, then you will get an output:

How to display "Hello World" in Python 3

Then type the following code:

print('This will teach you how to display "Hello World" in the programming language called Python.')

# You know how to type the rest! You don't need to type in the links or buttons.
