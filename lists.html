Lists (eg. techy, asheng)

Meet Techy and Asheng!

Today, we a going to make a list

First things first: download Python

Go to python.org > Download and follow the steps to set up Python on your computer

After you have Python installed, a new icon will appear on your desktop, labeled "Python 3.8." You click on it, and it opens the Python shell.

-----------------------------------------

Type in this code as your first line:

---------------

developers = [techy, asheng]

---------------

This makes a list called developers, consisting of two poeple: Techy(me) and Asheng.

Now let us add some stuff to the list

---------------

developers.append(notADeveloper)

developers.append(alsoNotADeveloper)

---------------

Let's print our list!

---------------

print(developers[0])

print(developers[1])

print(developers[2])

print(developers[3])

OUTPUT:

techy

asheng

notADeveloper

alsoNotADeveloper
