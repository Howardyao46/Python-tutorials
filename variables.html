Understanding Variables

Variables are like boxes that store stuff in them. You can assign a string or an integer to a variable. For example, if you type a = 38, now the variable a is 38. So that means, if you type print(a) then your output will be 38.

Python 3.8 -- o X

>>> a = 38

>>> print(a)

38
